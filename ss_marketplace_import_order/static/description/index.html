Marketplace Import Order for Community Edition v15

Indonesian Marketplace Platform

Create Your own Platform

You can add a new platform to import from any marketplace. But first you need to mapped the column from the excel, so odoo will mapped the right column to the right field.

Shipping Auto Debit

You can list the shipping courier from you marketplace that you don't want to import it to odoo. So if add it to "Shipping Auto Debit" and enabled it, that shipping courier won't be imported to odoo (shipping price won't be imported too)

Import Your Marketplace Order to Odoo

After finish your set up with your platform you can go to the Marketplace menu, to start uploading your marketplace order to Odoo.
